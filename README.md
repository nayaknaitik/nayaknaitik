#### 🤖 [Tagore Wisdom RAG Chatbot](https://github.com/nayaknaitik/tagoreNess-ChatBotAgent-Frontend)
*An AI-powered conversational assistant that delivers emotionally-aware, citation-backed wisdom from Rabindranath Tagore's complete works.*
* **Stack:** FastAPI, Python, Next.js 15, React 19, LangChain, FAISS-CPU, Groq Llama-3.3, TailwindCSS
* **Key Backend Achievements:** Implemented a context-aware intent extraction layer and a strict grounding validation loop to prevent LLM hallucinations. Tracked and deployed pre-computed FAISS vector database on Hugging Face Spaces using Git LFS, and connected it to a Next.js serverless proxy on Vercel.

#### 📨 Notification Service
*A high-throughput asynchronous messaging pipeline built to deliver notifications reliably at scale.*
* **Stack:** Node.js, Redis Streams, PostgreSQL, Docker
* **Key Backend Achievements:** Implemented consumer groups, Dead Letter Queue (DLQ) retry mechanisms, and request idempotency. Achieved **500 req/s throughput @ 180ms latency**.
